What is Nodejs

Node.js is an open source server environment;
 which helpus to write Javascript code in server

 What is NPM 

npm is the world's largest Software Registry.
The registry contains over 800,000 code packages.
Open-source developers use npm to share software.
Many organizations also use npm to manage private development.



To Install Nodejs
Step 1 : goto https://nodejs.org/en/download/
Step 2 : Download Window Installer and run and Install
(once we install we can use command start with node,npm)

open Commandline type 
node(Enter)

operable program or batch file.
